## Biblioteca

Sistema de Biblioteca echo en Laravel 5.5 con modulos de:

- Proveedores
- Compras
- Inventarios
- Editoriales
- Libros 
- Prestamos
- Devoluciones
- Empleados
- Usuarios

A si mismo incluye dos paneles uno para administradores y otro para usuarios.

## Frameworks y Librerias

- Laravel 5.5
- Bootstrap 4
- Bootstrap Select
- DataTables
- FontAwesome
- Animate.css
- Jquery
